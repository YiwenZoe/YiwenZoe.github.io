Leetcode 2370 Longest Ideal Subsequence

  • *** DFS ***
  • *** DP ***

DFS –> TLE

  • Start from base case
    • index == n-1: return 1
  • For each index, find the valid adjacent letters among 26 letters according to K
  • Choose the first valid adjacent letters in the string(smallest index)
    • Dictionary record the index of each letter in s
    • Choose the letter with the largest dfs value
  • The meaning of dfs function(what is the return value)
    • return start from index to end the length of the longest ideal subsequences

DP

  • Explanation

    • dp[c] means the length of the longest ideal subsequence ending with character c.

    • Iterate the character i in string s, c can be the next character for string ending from i - k to i + k.So that dp[i] = max(dp[i-k], dp[i-k+1] … , dp[i+k]) + 1.

    • return the max(dp) as result.

  • Complexity

    • Time O(n)
    • Space O(128)
