Leetcode 2376 Count Special Integers

  • Permutataions order matters: You have n items and want to find the number of ways k items can be ordered
    • A(n,k) = n!/(n-k)!
    • from itertools import permutations
  • Combinations order does not matter: number of ways to combine k items from a set of n
    • C(n,k) = A(n,k)/k! = n!/((n-k)! * k!)
