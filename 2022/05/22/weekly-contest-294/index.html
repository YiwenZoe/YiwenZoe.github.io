单调栈 + presum + presum of presum

  • https://leetcode.com/problems/sum-of-total-strength-of-wizards/discuss/2062997/Python-step-by-step-optimization-and-margin-decision

  • 思路

    • The basic idea is for each index i, we find the range such that nums[i] is the minimum value of this range. Then we simply added the subarray which contains nums[i] within the range: sum(subarray)*nums[i].
    • To avoid duplicate intervals, for each index i, only left or right can be <= nums[i].

  • Optimize step by step

    Step1

    • Now we have left and right of the subarray. Next we should find all subarrays in [left, right] which contains nums[i]
    • left index range from [left, i]
    • right index range from [i, right]
    • Then nums[i] * sum of subarrays

    Step2

    • We can simply get rid of loop j by presum of nums: presum

    Step3

    • Remove loop r
    • In r loop, presum[l] is the same, assume presum[l] = a
    • for r in range(i, right+1): nums[i] * (presum[r+1] - a) –> nums[i] * SUM(presum[i+1] + presum[i+2] + … + presum[right+1]) - nums[i] * a * (right + 1 - i)
    • presum of presum: pss –> pss[right+1] - pss[i] = SUM(presum[i] + presum[i+1] + … + presum[right+1])
    • for r in range(i, right+1): nums[i] * (pss[right+1] - pss[i]) - nums[i] * presum[l] * (right + 1 -i) = nums[i] * ((pss[right+1] - pss[i]) - presum[l] * (right + 1 -i))

    Step4

    • Remove loop l

  • pre sum

    • nums = [1,2,3,4,5]
    • presum = [0] 一开始先放一个0
    • presum[r+1] - presum[l] = nums[l] + nums[l+1] + … + nums[r]
