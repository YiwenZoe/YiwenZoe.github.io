Chapter2 Data Structure

Union Find

  1. 合并集合
    并查集
  • 近乎O(1)
  • 目的：1. 将两个集合合并 2. 询问两个元素是否在一个集合中
  • 基本原理：每个集合用一颗树来表示。树根的编号就是整个集合的编号。每个节点储存它的父节点。p[x]表示x的父节点。

问题：

  1. 如何判断数根
  • 当p[x] = x时，代表x为根节点
  1. 如何求x的集合编号
  • while p[x] != x: x= p[x]
  1. 如何合并两个集合
  • p[x] = y
  1. 连通块中点的数量
  • Initilize parent list: p[i] = i
  • Initilize parent list: size[i] = 1
  1. 食物链

  • 先说的不冲突的话为真话，后面与前面真话冲突的为假话

  • 把所有有关系的node放入一个集合中，树的结构

  • 用每个点到跟节点的距离表示这个node与跟节点的关系

  • 跟节点为第0代，所有第1代可以吃第0代，所有第2代可以吃第1代，所有第3代可以吃第2代… 0 -> 1 -> 2 -> 3 -> 4 … 第0代和第3代都可以吃第2代，第0代和第3代是同类，第1代和第4代是同类…

  • 由于是三种动物之间的关系, 每个node到跟节点的距离只有三种情况，所以可以用距离%3之后的值来定义

    • 0: 与跟节点同类
    • 1: 可以吃跟节点
    • 2: 可以被跟节点吃

  • 路径压缩，node到父节点的距离 -> node到跟节点的距离

  • 构建集合中node的关系,

    • if p[x] != p[y], 需要把两个node放入同一个集合中, p[p[x]] = p[y]
    • d[x]: distinct between x -> p[x]
    • d[y]: distance between y -> p[y]
    • ? : distance between p[x] -> p[y]
      1. if D = 1 x,y同类:
      • (dx - dy)%3 = 0 if p[x] == p[y]
      • d[x] + ?)%3 == d[p[y]]%3 -> (d[x] + ? - d[p[y]])%3 == 0 -> ? = d[y]-d[x], 这里其实就是让x到p[y]的距离等于y到p[y]的距离，因为x和y为同类
      1. if D = 2 x吃y:
      • x是2代动物，y是1代动物
      • (dx - dy)%3 = 1 if p[x] == p[y]
      • d[x] + ?)%3 == d[p[y]]%3 + 1 -> (d[x] + ? - d[p[y]] - 1)%3 == 0 -> ? = d[y]-d[x] + 1, 这里其实就是让x到p[y]的距离等于y到p[y]的距离 + 1，因为x吃y
