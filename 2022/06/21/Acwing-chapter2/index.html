Acwing 840. 模拟散列表

哈希表

  • 存储结构
    • 开放寻址法
    • 拉链法
  • 字符串哈希方式

与离散化的区别

  • 离散化是特殊的哈希方式
  • 离散化需要保序（单调）

什么情况下要用到哈希表

  • 很大的空间映射 (10^-9 - 10^9)–> 到小空间(0-10^5)
  • 哈希函数 h(x) -> mapping到（0-10^5)
    • x mod 10^5
    • 冲突：两个不一样的数映射为同一个数-> h(x) = h(y)
      • 处理冲突 1.拉链法 2.开放寻址法

拉链法

  • 开一个一维数组来存储所有的哈希值(假设h(x) mapping到 0-10^5-1)
  • 数组下标从 0 - 10^5 -1， 长度为10^5
  • 0 1 2 3 4 5 6 …
  • h(11) = h(23) = 3 ; 3->11->23
  • 每个槽上存储当前有的数
  • 哈希表为期望算法。平均来看每条链上的长度可以看成常数（很短）-> 时间复杂度O(1)
