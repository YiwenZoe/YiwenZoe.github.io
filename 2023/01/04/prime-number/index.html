质数

  1. 1

  2. 只包含1和本身两个约束

(1)质数的判定
试除法 O(sqrt(n))
n = a * b
n % a == 0 & n % (n/a) == 0
a <= b -> a <= n/a -> a <= sqrt(n)

(2)分解质因数
试除法 最好情况：logn / 最坏情况：O(sqrt(n))
核心思想：

  • 只需要遍历从2到sqrt(n)的所有整数就能得到质因数。
  • 当枚举到整数i，n当中不包含任何从[2, i-1]的质因子了，因为在之前的步骤里已经把i的质因子都除干净了。所以如果n%i ==0就代表i也不包含任何从[2, i-1]的质因子。
    所以i一定是之前没有出现过的质数。
  • n中最多只包含一个大于sqrt(n)的质因子，最后n分解质因子之后剩下的n如果>1, 则这个n就是那个大于sqrt(n)的质因子

(3)筛质数

  1. 朴素筛复杂度：n/2+n/3+…+n/n=n×(1/2+1/3+1/n)≈nlogn
    n→∞1+1/2+1/3+1/n≈lnn+C C(Eulerconstant)≈0.577C(Eulerconstant)≈0.577

  2. 埃式筛法（只筛素数）复杂度：nloglogn
    质数定理：[1,n]中有n/lnn个质数
    x→∞π(x)≈nlnnx→∞π(x)≈nln⁡n π(x)π(x)表示不大于x的素数的个数

  3. 线性筛法
    核心思想：n只会被它的最小质因子筛掉

  • i % primes[j] == 0: primes[j]一定是i的最小质因子， 并且primes[j]一定是primes[j] * i的最小质因子
  • i % primes[j] != 0: primes[j]一定小于所有i的质因子，并且primes[j]也一定是primes[j] * i的最小质因子

对于一个合数x, 假设rimes[j]是x的最小质因子，当i枚举到x/primes[j]的时候，x一定会被筛掉。
每个数只用最小质因子筛，每个数只有一个最小质因子，每个数只会被筛一次，所以是线性的。

// 为了保证 iprimes[j] 会且只会 被 它的最小质因数筛掉，这里需要证明
// primes[j] 是 iprimes[j] 的 最小质因数
// Question 1. 当 i % primes[j] == 0 时，primes[j] 是 primes[j]*i 的最小质因数吗？
// Answer: 是，由于primes的递增性以及break, 保证了 i碰到 i % primes[j] == 0 时, primes[j] 最小
//
// Question 2: 为什么i%primes[j] == 0 时要停止？有没有可能存在一个数 t = primes[k]*i,其中 k>j
// 他的最小质因数也是 primes[j]？
// Answer: 如果不停， primes的下标从 j 继续往后走，t 第一次会被 primes[k] 筛掉，而不是 primes[j]， 具体点说
// 假如不停，前进到 primes[j+1] 则有 t = primes[j+1]*i
// 而 i = primes[j]*k, 所以 t = primes[j+1]primes[j]k = primes[j](kprimes[j+1) = primes[j]*k’ 可见 t 的最小质因数确实是primes[j]
// 但是被 primes[j+1]筛了，并且 当 i 到 t 时，又会被 primes[j] 筛一遍，所以重复了
//
// Question 3: i%primes[j] != 0 时，primes[j] 是不是 primes[j]*i 的最小质因数？
// Answer: 当为i遍历primes时，存在 h 使得 i % primes[h] == 0, h < j, 那由于 break那一行，会立即停止，并不会走到j，假如i遇到了j，
// 说明当前没有遇到 任意一个h使得 i % primes[h] == 0, 或者说，所有 j 之前的质数 都无法被 i 整除 那自然也无法被 primes[j]*i 整除
// 所以primes[j]一定是 primes[j]*i 的最小质数
